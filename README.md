## Conways Game Of Life

### Install Dependencies 
<code> npm install </code>

### Build 
<code> npm run build </code>

### Run in dev
<code> npm run dev </code>

### Run in prod 
<code>npm start</code>


### View in dev 
<code>http://localhost:3000</code>

### View in prod 
<code> https://conwaysgame-of-life.herokuapp.com </code>
